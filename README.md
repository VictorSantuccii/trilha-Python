# 🐍 Trilha de Estudos: Python

Bem-vindo(a) à **Trilha de Estudos em Python**, onde você vai aprofundar de forma progressiva e prática na linguagem que está em **alta** no mercado, tanto para **back-end**, **automação**, **data science** quanto **APIs e scripts inteligentes**.

---

## 🎯 Objetivo

> Dominar o Python moderno para:
- Desenvolver scripts e automações 
- Criar APIs RESTful com Flask e FastAPI
- Trabalhar com banco de dados (PostgreSQL, SQLite)
- Fazer scraping, manipulação de dados e análise
- Criar testes automatizados
- Deploy de aplicações com Docker
- Conhecer estruturas modernas de projetos com boas práticas

---

## 🧱 Estrutura da Trilha

| Semana | Tema Principal                           | Conteúdo                                                                 |
|--------|-------------------------------------------|--------------------------------------------------------------------------|
| 01     | Fundamentos da linguagem Python           | Sintaxe, tipos, funções, classes                                         |
| 02     | Programação Orientada a Objetos (POO)     | Herança, encapsulamento, abstração, polimorfismo                        |
| 03     | Manipulação de arquivos e automações      | Leitura/escrita, JSON, CSV, agendamentos, scripts automatizados         |
| 04     | Web Scraping + Requests + BeautifulSoup   | Requisições HTTP, parsing de páginas, scraping inteligente              |
| 05     | Criação de APIs com Flask e FastAPI       | Rotas, métodos HTTP, JSON, middlewares, validação com Pydantic          |
| 06     | Banco de dados com SQLAlchemy             | CRUD, migrations, relações, integração com PostgreSQL                   |
| 07     | Testes automatizados                      | `unittest`, `pytest`, cobertura, mock                                   |
| 08     | Deploy com Docker + Heroku + Railway      | Dockerfile, Compose, deploy, configs com `.env`, produção               |

## 🐣 Semana 01 — Fundamentos da Linguagem

📁 `semana-01-fundamentos/`

### ✅ Conteúdo:
- Sintaxe básica, tipos primitivos, estruturas de decisão e repetição
- Funções, módulos, imports
- Listas, dicionários, tuplas, sets
- Boas práticas com `pep8` e docstrings

---

## 🧱 Semana 02 — Programação Orientada a Objetos

📁 `semana-02-poo/`

### ✅ Conteúdo:
- Classes, objetos e construtores
- Herança, polimorfismo, encapsulamento
- Métodos mágicos (`__init__`, `__str__`, etc)
- Abstração com classes abstratas (`abc`)

---

## ⚙️ Semana 03 — Automação e Arquivos

📁 `semana-03-automacoes/`

### ✅ Conteúdo:
- Leitura e escrita de arquivos `.txt`, `.json`, `.csv`
- Scripts de automação: renomear arquivos, agendar tarefas, envio de emails
- Uso de `os`, `shutil`, `schedule`

---

## 🌐 Semana 04 — Web Scraping

📁 `semana-04-web-scraping/`

### ✅ Conteúdo:
- Requests com `requests`
- Extração de dados com `BeautifulSoup` e `lxml`
- Criação de robôs de coleta de dados
- Armazenamento dos dados em arquivos ou banco

---

## 🚀 Semana 05 — APIs com Flask e FastAPI

📁 `semana-05-apis/`

### ✅ Conteúdo:
- Criação de APIs REST com Flask e FastAPI
- Rotas GET, POST, PUT, DELETE
- Validação com Pydantic (FastAPI)
- Middlewares, JWT e autenticação simples

---

## 🗃️ Semana 06 — Banco de Dados com SQLAlchemy

📁 `semana-06-sqlalchemy/`

### ✅ Conteúdo:
- Configuração com PostgreSQL e SQLite
- Models e migrations
- Relacionamentos e operações CRUD
- Integração com APIs

---

## 🧪 Semana 07 — Testes Automatizados

📁 `semana-07-testes/`

### ✅ Conteúdo:
- Testes com `unittest` e `pytest`
- Testes de APIs com `TestClient` (FastAPI)
- Mocks, fixtures e cobertura de testes
- TDD básico

---

## ☁️ Semana 08 — Deploy com Docker

📁 `semana-08-deploy/`
